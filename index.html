مواقع تعلم البرمجة

ابدأ رحلتك في البرمجة من الصفر حتى الاحتراف

  • Scratch (MIT) — البرمجة بالكتل، ممتاز للمبتدئين
  • Code.org — Hour of Code وحلقات تعلّم متدرّجة
  • Tynker — مشاريع برمجية تفاعلية
  • Khan Academy — مقدمة في البرمجة بـ JavaScript
  • Microsoft MakeCode — برمجة الروبوتات والألعاب
  • Codecademy — مسار Python للمبتدئين
  • Khan Academy — Computer Programming
  • freeCodeCamp.org — مسارات شاملة
  • Replit — تجربة Python مباشرة في المتصفح
  • Udacity — مسارات مهنية متخصصة
🎨
مواقع التصميم ثلاثي الأبعاد

أدوات التصميم الهندسي والطباعة ثلاثية الأبعاد

  • Tinkercad (Autodesk) — الأنسب لهذا العمر، يعمل في المتصفح
  • 3D Slash — واجهة لعبة بسيطة وممتعة
  • BlocksCAD — تصميم بالبرمجة بالكتل
  • SketchUp Free — للتصميم المعماري البسيط
  • ابدأ بتصميم: مفتاح، حامل قلم، لعبة بسيطة
  • Onshape — CAD سحابي احترافي مجاني للطلاب
  • Fusion 360 — مجاني للطلاب والمعلمين
  • Tinkercad — للبدء السريع قبل Onshape
  • FreeCAD — خيار مفتوح المصدر
  • ابدأ بمشروع متكامل: تصميم نموذج لمشكلة هندسية
